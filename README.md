## ⚡ Getting Started — Initiate Hyperspace Jump

```bash
# Clone the Academy archives
git clone https://github.com/chrisvdalen/spring-boot-blueprints.git
cd spring-boot-blueprints

# Each module is self-contained — navigate to any sector
cd api-design/rest-conventions

# Initiate engines
./mvnw spring-boot:run

# The API docs shall appear at:
# http://localhost:8080/swagger-ui.html
```

> **Prerequisite:** Java 25 or higher. The Force does not support Java 8.

## 🤝 Contributing to the Academy

> *"The Jedi Code is not a closed system. It evolves. So too must this repository."*

Pull requests are welcome, young Padawan. Please ensure:

1. Each module has a `README.md` explaining the **WHY**
2. Tests pass (`./mvnw verify`)
3. Your code uses Java records where appropriate (no Lombok classes for DTOs)
4. Constructor injection — never field injection (`@Autowired` on a field is the dark side)
5. Anti-patterns are clearly labelled with `// ANTI-PATTERN:` comments
